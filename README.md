# Lume Theme Shiraha

## Usage

```ts
// _config.ts
import lume from 'lume/mod.ts'
import shiraha from 'https://raw.githubusercontent.com/importantimport/lume_theme_shiraha/main/mod.ts'

export default lume().use(shiraha())
```
